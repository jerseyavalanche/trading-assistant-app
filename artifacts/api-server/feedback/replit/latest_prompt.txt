Implement a race-ranking fairness fix for the AI Brain 4-bot race. Also add tests covering: identical equity after first cycle with zero PnL, identical equity after mark-to-market with real price movement, and a session where some bots are ineligible due to market hours. Keep existing data structures where possible, but add explicit fields if needed to represent 'comparison deferred' and 'excluded from ranking'.
